# Railway GitHub 연결 문제 해결 가이드

## 문제 상황
- `fatal: could not read Username for 'https://github.com': terminal prompts disabled`
- GitHub repository 접근 권한 에러

## 해결 방법

### 1. Railway 콘솔에서 GitHub 재연결
1. Railway 대시보드 → 프로젝트 선택
2. Settings → GitHub 탭
3. "Reconnect Repository" 클릭
4. GitHub 권한 재승인

### 2. GitHub Personal Access Token 확인
- GitHub Settings → Developer settings → Personal access tokens
- Repository 접근 권한 확인 (repo, workflow, admin:org)
- 만료일 확인 및 갱신

### 3. Repository Secrets 설정 (필요시)
- GitHub Repository → Settings → Secrets and variables → Actions
- RAILWAY_TOKEN 또는 GITHUB_TOKEN 확인

### 4. 임시 해결: Railway CLI 수동 배포
```bash
railway login
railway link [project-id]
railway up
```
